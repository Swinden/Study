# 最短路径问题

**线段无长度**

**示例**：使用字典的方式构建有向图，并搜索图中的路径。

图很容易通过列表和词典来构造。比如说，这有一张简单的图：

　A -> B
 A -> C
 A -> D
 B -> E
 C -> D
 C -> F
 D -> B
 D -> E
 E ->  
 F -> D
 F -> G
 G -> E

 graph = {'A': ['B', 'C','D'],
        'B': [ 'E'],
        'C': ['D','F'],
        'D': ['B','E','G'],
        'E': [],
        'F': ['D','G']
        'G': ['E']}
